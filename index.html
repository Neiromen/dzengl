Детский телефон доверия

Проект по созданию службы психологической помощи для детей и подростков

О проекте

Детский телефон доверия — это бесплатная анонимная служба психологической помощи детям и подросткам, оказавшимся в трудной жизненной ситуации. Наша цель — создать безопасное пространство, где каждый ребенок может получить поддержку, понимание и помощь квалифицированных специалистов.

Конфиденциально

Все обращения строго анонимны. Мы гарантируем полную конфиденциальность каждого разговора.

Профессионально

На линии работают психологи с профессиональным образованием и специальной подготовкой.

Доступно

Работаем круглосуточно, 7 дней в неделю. Звонок с любого телефона бесплатный.

Ответы на вопросы

1. Что такое ChildLine?
ChildLine — это британская бесплатная круглосуточная служба психологической помощи детям и подросткам, которая предоставляет конфиденциальные консультации по телефону и онлайн.
2. Где она находится?
Штаб-квартира ChildLine находится в Лондоне, но служба работает по всей Великобритании.
3. Когда она была создана?
ChildLine была основана в 1986 году журналисткой Эстер Ранцен.
4. Кому она помогает?
Служба помогает детям и подросткам до 19 лет, оказавшимся в сложных жизненных ситуациях.
5. Какие проблемы помогают решать консультанты?
Консультанты помогают с проблемами издевательств в школе, семейных конфликтов, психологического насилия, депрессии, экзаменационного стресса и многих других.

Наш проект: "Детский телефон доверия"

Когда создан

Проект был основан в 2023 году в ответ на растущую потребность в психологической помощи детям и подросткам.

Кому помогаем

Мы оказываем помощь детям и подросткам от 6 до 18 лет, а также консультируем родителей и педагогов.

Как помогаем

Предоставляем круглосуточные телефонные консультации, онлайн-чат, информационные материалы и групповые вебинары.

Волонтерская помощь

Наш проект активно привлекает волонтеров для различных задач:

  • Подготовка и распространение информационных материалов
  • Организация благотворительных мероприятий
  • Административная поддержка
  • Помощь в проведении онлайн-мероприятий

Как развернуть этот сайт на GitHub

Пошаговая инструкция:

  1. Создайте аккаунт на GitHub (если у вас его еще нет) на сайте github.com
  2. Создайте новый репозиторий: нажмите кнопку "+" в правом верхнем углу и выберите "New repository"
  3. Назовите репозиторий: например, "child-helpline-project"
  4. Выберите публичный доступ (Public), чтобы сайт был доступен всем
  5. Добавьте файлы:
    • Создайте файл с именем index.html
    • Скопируйте весь HTML-код этого сайта в этот файл
    • Сохраните файл
  6. Включите GitHub Pages:
    • Перейдите в настройки репозитория (Settings)
    • В левом меню выберите "Pages"
    • В разделе "Source" выберите ветку "main" и папку "/ (root)"
    • Нажмите "Save"
  7. Дождитесь публикации: через несколько минут ваш сайт будет доступен по адресу: https://[ваш_логин_github].github.io/[название_репозитория]

Альтернативный способ (через загрузку файлов):

  1. Сохраните этот HTML-код в файл с именем index.html на вашем компьютере
  2. Зайдите в ваш репозиторий на GitHub
  3. Нажмите кнопку "Add file" → "Upload files"
  4. Перетащите файл index.html в окно браузера или выберите его через проводник
  5. Нажмите "Commit changes"
  6. Активируйте GitHub Pages в настройках репозитория как описано выше

Важно: имя файла должно быть именно index.html, так как это стандартное имя для главной страницы сайта.
